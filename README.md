# IrrigaTech 

Projeto acadêmico de automação de irrigação desenvolvido no Entec 2024 evento da Universidade de Uberaba - UNIUBE.

## Descrição
Sistema de irrigação automática que ajusta a quantidade de água com base
em dados de sensores de umidade do solo, utilizando ESP32 e Python,
com foco em uso eficiente da água.

## Tecnologias Utilizadas
- Python
- ESP32
- Arduino Uno
- Sensores de umidade do solo
- Protocolo MQTT
- Automação de válvulas

## Funcionamento do Sistema
- Coleta dados do solo por sensores
- Processa as informações
- Decide automaticamente quando irrigar
- Ativa válvulas e envia notificações

## Resultados
- Redução do desperdício de água
- Automação do processo de irrigação
- Aplicação prática de conceitos de IoT e programação
